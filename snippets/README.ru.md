# Сниппеты

Коллекция наборов сниппетов для [Raycast](https://www.raycast.com/).

## Как импортировать

1. Откройте Raycast
2. Наберите "Import Snippets"
3. Выберите нужный `.json` файл
4. Готово — сниппеты доступны для использования

### Claude Code

Количество: 18 | Файл: [`claude-code-snippets.json`](claude-code-snippets.json)

Шаблоны рабочих промптов для [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Наберите триггер и получите структурированный промпт с плейсхолдером `{cursor}` для вашего ввода.

Триггер: `/keyword`

| Триггер | Описание |
|---------|----------|
| `/feature` | Новая фича (полный цикл разработки) |
| `/bugfix` | Исправление бага с тестом |
| `/refactor` | Рефакторинг с покрытием тестами |
| `/newproject` | Новый проект с нуля |
| `/tests` | Добавить тесты без изменения кода |
| `/review` | Чеклист код-ревью |
| `/onboard` | Быстрое знакомство с проектом |
| `/docs` | Написание документации |
| `/perf` | Оптимизация производительности |
| `/deploy` | Docker и CI/CD |
| `/cleanup` | Очистка мёртвого кода и артефактов |
| `/security` | Аудит безопасности (OWASP Top 10) |
| `/migrate` | Безопасная миграция БД |
| `/api` | Проектирование REST API |
| `/quickfix` | Минимальное точечное исправление |
| `/explain` | Объяснение работы кода |
| `/plan` | Создать план без кодинга |
| `/continue` | Продолжить с места остановки |
